Day 1
=====

Overview
--------
Day 1 will establish fundamental skills using the command line and the version control software, `git`.
These are essential tools for software developers in any field, and you will use and expand your use of them throughout the MSSE degree.
The command line and git have high learning curves, but open up a world of possibilities once you learn them.
For many poeple, the command line becomes a preferred method for using a computer.
